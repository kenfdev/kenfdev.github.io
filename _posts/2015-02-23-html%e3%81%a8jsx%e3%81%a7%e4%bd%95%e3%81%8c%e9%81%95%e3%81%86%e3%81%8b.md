# HTMLとJSXで何が違うか

[JSX Gotchas][1]に書かれているように、JSXとHTMLはちょっぴり違いがあります。大きくは次の二個のようですね。  
* HTMLエンティティ  
* HTMLカスタム属性

## HTMLエンティティの注意点

以下、gistにまとめてみました

## HTMLカスタム属性の注意点

カスタム属性は標準ではコンパイルされないので注意が必要です。カスタム属性を使う時はかならずdataをprefixにつけましょう。

  <noscript>

  </noscript>
</div>
